About ThingiRec

ThingiRec suggests similar users with whom users of the site Thingiverse.com should connect. Recommendations are created by comparing the cosine similarity between item descriptions vectorized by tf-idf. ThingiRec was created by Robert Senseman as a capstone project for the Galvanize Data Science Immersive program. Project detail can be seen in the project Github repo. {% endblock %}
